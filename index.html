/!\ ATTENTION /!\ : Les données sont stockées dans votre navigateur, si vous changez de navigateur, que vous supprimez ses données ou que vous changez d'ordinateur, les données seront perdus

Comment personnaliser ma liste de devoirs à faire ?

  1. Cliquer sur l'icone data au niveau de l'entête
  2. Copier les lignes suivantes en personnalisant le titre, la date et la valeur isDone :
    3. {
    "title": "S.V.T #1",
    "endingDate": {
        "forecast": "26 sept 2019",
        "real": "27 sept 2019"
    },
    "isDone": true
},
  3. Coller les lignes après un autre bloc du même type
  4. Enregistrer vos modifications
  5. Baf ! Magie ! Normalement ça marche

/!\ ATTENTION /!\ : la virgule après le bloc de code doit être présente à tous les blocs sauf sur le dernier !

Comment copier mes données d'un navigateur à un autre ?

  1. Ouvrez la page sur le premier navigateur
  2. Cliquer sur l'icone data au niveau de l'entête
  3. Copier tout le contenu
  4. Ouvrez la page sur le premier navigateur
  5. Cliquer sur l'icone data au niveau de l'entête
  6. Coller le contenu de votre presse-papier
  7. Enregistrer vos modifications
